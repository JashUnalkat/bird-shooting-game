# 🐦 Bird Shooting Mobile Game

---

## 📖 Project Overview

Bird Shooting Mobile Game is a cross-platform 2D mobile game developed using **React Native**, **Expo**, and the **Matter.js** physics engine. The game challenges players to improve their reflexes and accuracy by aiming and shooting moving birds while earning points throughout gameplay.

The project demonstrates the implementation of mobile game development concepts including physics simulation, collision detection, animations, touch interactions, score management, and responsive game mechanics.

Designed with a simple and engaging interface, the application focuses on delivering smooth gameplay while showcasing object-oriented programming concepts and real-time physics simulation.

## 📱 Game Flow

1. Launch the application
2. Start a new game
3. Birds begin moving across the screen
4. Tap to shoot moving birds
5. Score increases with successful hits
6. Continue playing to achieve a higher score
7. View final score
8. Restart and play again

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/JashUnalkat/bird-shooting-game.git
```

Navigate to the project folder

```bash
cd bird-shooting-game
```

Install dependencies

```bash
npm install
```

Run the application

```bash
npx expo start
```

Open the application using:

- Expo Go
- Android Studio Emulator
- Physical Android Device
