This is application made by Dinh Huy Nguyen and Jash Unalkat for Final Project for INFO-5144 course. This code is created by Visual Studio Code and snack.expo.dev. 

The idea of the game from the combination between 2 most popular game from our childhood. Chicken Shooter and Flappy Bird. And especially, the Flappy Bird game was created in Vietnam, the hometown of Dinh Huy Nguyen. 

Still keeping the simple playstyle from those two games, we using the finger to move the bird up down how to avoid the pipes or the enemies, each pipe passed, we gain 1 point, the bird can shoot the enemies to get extra points when killed the enemies. When bird got the fire, the can shoot the bullets with wider range in 5 seconds.
